## Features

- Sign In
- Dashboard 
- DataTables
- Error Pages
- Light & Dark Mode
- Error Handling
- Network Connection Check
- Responsive Design



## Tech Stack

**Client:** TypeScript, React, React Redux, RTK Query, ShadcnUI, Formik, Yup 

**API:** Platzi Api



## Deployment

Package Installation

```bash
  npm run install
```

Running with development server

```bash
  npm run dev
```

Building production

```bash
  npm run build
```
